# Atlas AI Architecture v0.1

## Core Services

- Prompt Library
- AI Runner
- Response Store
- Intelligence Engine
- Scoring Engine
- Report Generator
- Dashboard

## Data Philosophy

Raw responses are never overwritten. Every run becomes historical evidence that can support trend analysis later.

- Python
- SQLite

- CSV seed data

## Future Stack

- PostgreSQL
- FastAPI

- Background scheduler
