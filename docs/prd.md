# Atlas AI Product Requirements Document v0.1

## Mission

Atlas AI monitors how AI models understand a market, compares brands and competitors, detects changes over time, and recommends actions to improve visibility and positioning.

## MVP Questions

1. How often is Firman mentioned?
2. Which competitors are mentioned instead?
3. Which features and use cases are associated with each brand?
4. Which prompts create visibility gaps?
5. What opportunities should be prioritized?

## MVP Market

Portable Power.

## MVP Users

- Marketing
- Product Management
- Executive Leadership
- Competitive Intelligence
- SEO / Content teams

## MVP Workflow

1. Load prompt library.
2. Collect AI responses.
3. Store raw responses.
4. Extract brands and features.
5. Score visibility.
6. Produce dashboard and report.

## Non-goals for v0.1

- User login
- SaaS billing
- Multi-tenant permissions
- Fully automated web scraping
- Production API orchestration
